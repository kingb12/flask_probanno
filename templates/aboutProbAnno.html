ProbAnno (Probabilistic Annotation) is a method for annotating each protein sequence in a proteome based on similarity to proteins of known function in other organisms, and assigning a probability to each annotation.

About

  • Initial release: October 1, 2016
  • Report an issue with the web-site or API
