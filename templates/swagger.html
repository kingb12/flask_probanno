Query parameters

  "status" : "Complete"
}

get /model/gapfill
probabilistically gap-fill a metabolic model (gapfillModel)
Run a probabilistic gap-filling with a model

Request headers

Query parameters

model_id (required)
Query Parameter — metabolic model to gapfill
output_id (required)
Query Parameter — output ID for the gap-filled metabolic model
fasta_id (required)
Query Parameter — reaction likelihoods ID to inform gap-filling
template (optional)
Query Parameter — Template model to inform gap-filling

Return type

Job

Example data

Content-Type: application/json
{
  "jid" : "a70a6344-3c55-48fa-8b60-db71858e810b",
  "job" : "calculate_probanno",
  "target" : "my_model",
  "status" : "Complete"
}

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

successful operation Job

400

No session, bad session, or missing arguments

404
